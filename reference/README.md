# Reference — 参考資料・リンク集

このフォルダに資料・リンク・PDFなどを入れると、Claudeがその内容をもとに学習コンテンツ（テキスト・クイズ）を生成します。

---

## フォルダ構成

```
reference/
  README.md              ← このファイル（インデックス）
  links.md               ← 参考URLのリスト
  statistics/            ← 統計検定関連の資料
  data-science/          ← データサイエンス関連の資料
  pdfs/                  ← PDF資料（置き場所）
```

---

## 資料の追加方法

### URLを追加する場合
`links.md` にURLとメモを書いてください。

### PDFを追加する場合
`pdfs/` フォルダにファイルを置いてください。

### メモ・ノートを追加する場合
該当カテゴリのフォルダに `.md` ファイルで置いてください。

---

## コンテンツ生成の流れ

```
① referenceに資料・リンクを追加
        ↓
② Claudeに「〇〇のテキストを作って」と依頼
        ↓
③ src/content/ に学習テキストが生成される
        ↓
④ Claudeに「〇〇のクイズを作って」と依頼
        ↓
⑤ src/quizzes/ にクイズJSONが生成される
```
