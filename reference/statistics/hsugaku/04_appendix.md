## 各種統計量の本来の定義
URL: https://hsugaku.com/32-10

**χ²分布（カイ二乗分布）の定義：**
- Z₁, Z₂, ..., Zₙ が互いに独立に N(0,1) に従うとき
- X = Z₁² + Z₂² + ... + Zₙ² 〜 χ²(n)

**t分布の定義：**
- Y〜χ²(n)、Z〜N(0,1) が独立のとき
- X = Z / √(Y/n) 〜 t(n)

**F分布の定義：**
- Y〜χ²(m)、Z〜χ²(n) が独立のとき
- X = (Y/m) / (Z/n) 〜 F(m, n)

## なぜ標本から作られた統計量は自由度が下がるのか
URL: https://hsugaku.com/32-20

- 標本分散が χ²(n-1) に従う理由（自由度が n でなく n-1 な理由）
- n=2 の特殊ケースで直感的説明：2点の偏差は1つ決まれば他は決まる
- 数学的証明：偏差平方和を独立な標準正規変数の二乗和として表現
  - Σ(Xᵢ - X̄)² は n-1 個の独立な成分に分解できる
- 「標本平均を推定することで自由度が1つ消費される」という解釈
