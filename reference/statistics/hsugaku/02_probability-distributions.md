URL: https://hsugaku.com/17-10

## #18 確率変数の分散
URL: https://hsugaku.com/18-10

- 分散 V(X) = E[(X - μ)²] = E(X²) - {E(X)}²
- 標準偏差 SD(X) = √V(X)
- 計算の実践：具体的な確率分布からの分散計算

## #19 確率変数の標準化
URL: https://hsugaku.com/19-10

- 標準化変換 Z = (X - μ) / σ
- 標準化後：E(Z) = 0、V(Z) = 1
- 異なる分布を比較するための共通スケールへの変換

## #20 同時分布
URL: https://hsugaku.com/20-10

- 2変数の同時確率分布（結合分布）
- 周辺分布：一方の変数について合計（積分）した分布
- 共分散 Cov(X,Y) = E(XY) - E(X)E(Y)
- 独立性：P(X=x, Y=y) = P(X=x)P(Y=y)

## #21 確率変数の和の期待値と分散
URL: https://hsugaku.com/21-10

- 期待値の線形性：E(X+Y) = E(X) + E(Y)（常に成立）
- 独立なとき：V(X+Y) = V(X) + V(Y)
- 独立でないとき：V(X+Y) = V(X) + V(Y) + 2Cov(X,Y)

## #22 ベルヌーイ分布
URL: https://hsugaku.com/22-10

- 結果が成功（1）か失敗（0）の2値の分布
- パラメータ p（成功確率）
- E(X) = p、V(X) = p(1-p)
- 二項分布の基礎となる分布

---

## #23 二項分布
URL: https://hsugaku.com/23-10

- n 回のベルヌーイ試行で成功が k 回の確率：B(n,p)
- P(X=k) = nCk × p^k × (1-p)^(n-k)
- E(X) = np、V(X) = np(1-p)
- n が大きくなると正規分布に近似できる

---

## #24 ポアソン分布
URL: https://hsugaku.com/24-10

- まれな事象の発生回数を表す分布：Po(λ)
- P(X=k) = e^(-λ) × λ^k / k!
- E(X) = λ、V(X) = λ
- 二項分布 B(n,p) で n→∞、p→0、np→λ のときの極限
- 応用例：単位時間あたりの到着客数・欠陥品数

---

## #25 幾何分布
URL: https://hsugaku.com/25-10

- 初めて成功するまでの試行回数の分布：Ge(p)
- P(X=k) = (1-p)^(k-1) × p（k回目で初めて成功）
- E(X) = 1/p、V(X) = (1-p)/p²
- 無記憶性：過去の失敗回数に関わらず次の成功確率は同じ

## #26 連続型確率変数
URL: https://hsugaku.com/26-10

- 連続型確率変数の確率密度関数 f(x) の定義
- P(a ≤ X ≤ b) = ∫[a to b] f(x)dx（面積が確率）
- ∫[-∞ to ∞] f(x)dx = 1（全確率は1）
- 累積分布関数 F(x) = P(X ≤ x)

---

## #27 連続型確率変数の平均や分散
URL: https://hsugaku.com/27-10

- 期待値 E(X) = ∫[-∞ to ∞] x f(x)dx
- 分散 V(X) = ∫[-∞ to ∞] (x-μ)² f(x)dx = E(X²) - {E(X)}²
- 一様分布 U(a,b)：E(X) = (a+b)/2、V(X) = (b-a)²/12

## #28 正規分布
URL: https://hsugaku.com/28-10 / https://hsugaku.com/28-20

- 正規分布 N(μ, σ²) の確率密度関数
- 左右対称の釣鐘型（ベル曲線）
- 正規分布の重要な性質：
  - μ±σ の範囲に約68%
  - μ±2σ の範囲に約95%
  - μ±3σ の範囲に約99.7%
- 線形変換の閉包性：X〜N(μ,σ²) → aX+b〜N(aμ+b, a²σ²)
- 標準正規分布 N(0,1) への変換方法

## #30 大数の法則と中心極限定理
URL: https://hsugaku.com/30-10 / https://hsugaku.com/30-20

**大数の法則：**
- 標本サイズ n が大きくなると標本平均 X̄ は母平均 μ に収束
- 弱大数の法則：確率収束

**中心極限定理：**
- n が十分大きいとき、X̄〜N(μ, σ²/n) に近似できる
- 元の分布の形によらず成立（n≥30 程度が目安）
- 統計的推測（推定・検定）の理論的基礎

## #32 歪度と尖度
URL: https://hsugaku.com/32-10 / https://hsugaku.com/32-20（おまけ扱い）

- 歪度（Skewness）：分布の左右非対称性を表す指標
  - 歪度 > 0：右裾が長い（右歪み）
  - 歪度 < 0：左裾が長い（左歪み）
- 尖度（Kurtosis）：分布の尖り具合を表す指標
  - 正規分布の尖度 = 0（超過尖度）
  - 尖度 > 0：正規分布より尖っている（裾が厚い）
  - 尖度 < 0：正規分布より平坦
