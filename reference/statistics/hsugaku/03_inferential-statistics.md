## #36 推定量の基準
URL: https://hsugaku.com/36-10

- 不偏性：推定量の期待値が母数に一致（偏りがない）
- 一致性：標本サイズが大きくなると母数に確率収束
- 有効性：不偏推定量の中で分散が最小
- 不偏推定量の例：標本平均・不偏分散（分母 n-1）

## #37 標本平均の分布
URL: https://hsugaku.com/37-10

- 正規母集団 N(μ, σ²) からの標本平均：X̄〜N(μ, σ²/n)
- 標準誤差（SE）= σ/√n：標本平均のばらつきの指標
- 母分散 σ² が既知の場合の標準化：Z = (X̄ - μ)/(σ/√n)〜N(0,1)

---

## #38 不偏分散の分布
URL: https://hsugaku.com/38-10

- 不偏分散 S² = Σ(Xᵢ - X̄)²/(n-1)
- (n-1)S²/σ² 〜 χ²(n-1)（カイ二乗分布に従う）
- なぜ分母が n-1 か：自由度の概念
- 母分散 σ² が未知のとき：t分布を使った推論

## #39 区間推定の一般的な手順
URL: https://hsugaku.com/39-10

- 信頼区間の定義：「母数を95%の確率で含む区間」ではなく「この手順で作った区間の95%が母数を含む」
- STEP1：点推定量を求める
- STEP2：推定量の分布を確認（正規・t・χ²・F）
- STEP3：信頼係数（1-α）に対応する臨界値を求める
- STEP4：信頼区間を構成する

|---------|------|---------|-------|

## #41 よく使う区間推定一覧（2標本）
URL: https://hsugaku.com/25-10

- **2標本の母平均差の区間推定**：
  - 分散既知：正規分布を使用
  - 等分散仮定：プール分散を用いた t(nₐ+nᵦ-2) 分布
  - 分散未知・不等：ウェルチの近似自由度
- **母分散比の推定**：F分布を使用
- **2標本の母比率差**：正規近似を使用
- 実務では区間推定より検定が多く行われる

## #42 標本の大きさの決定
URL: https://hsugaku.com/24-20

- 信頼区間の幅が指定値以下になる標本サイズ n を求める
- 母比率の場合：p の事前情報なし → p=0.5 を代入（最も保守的な推定）
- 実例：A県の洗剤使用世帯比率調査
  - 母比率不明：n ≥ 1,540 世帯
  - 母比率 ≤ 25%：n ≥ 1,160 世帯

## #43 仮説検定の一般的な手順
URL: https://hsugaku.com/26-10

- **STEP 0**：有意水準 α を設定（例：5%、1%）
- **STEP 1**：帰無仮説 H₀ と対立仮説 H₁ を設定
  - H₀：「母数 = 具体的な値」の形
  - H₁：片側検定（> or <）または両側検定（≠）
- **STEP 2**：H₀ のもとで検定統計量の分布を確認
- **STEP 3**：棄却域を設定 → 実現値を計算 → 結論
  - 実現値 ∈ 棄却域 → H₀ 棄却（H₁ を採択）
  - 実現値 ∉ 棄却域 → 「統計的に有意差なし」

## #44 よく使う仮説検定一覧（1標本）
URL: https://hsugaku.com/27-10

| 検定対象 | 条件 | 使用分布 | 統計量 |
|---------|------|---------|-------|
| 母平均 μ | σ² 既知 | N(0,1) | Z = (X̄-μ₀)/(σ/√n) |
| 母平均 μ | σ² 未知 | t(n-1) | T = (X̄-μ₀)/(S/√n) |
| 母分散 σ² | μ 未知 | χ²(n-1) | χ² = (n-1)S²/σ₀² |
| 母比率 p | n 大 | N(0,1) | Z = (p̂-p₀)/√(p₀(1-p₀)/n) |

実例：牛丼屋の盛り付け機械が260g超か（t検定）

## #45 よく使う仮説検定一覧（2標本）その1
URL: https://hsugaku.com/28-10

- **母平均差の検定**：
  - 分散既知：Z 統計量（正規分布）
  - 等分散仮定：プール分散使用、t(nₐ+nᵦ-2)
  - 分散未知・不等：ウェルチの検定（近似自由度）
- **母分散比の検定（F検定）**：
  - 統計量 F = S₁²/S₂²（大きい方を分子に）
  - F(nₐ-1, nᵦ-1) 分布に従う
- **母比率差の検定**：正規近似

実例1：大学院入試成績の比較（5名 vs 6名）
実例2：ハンバーガー重量のばらつき検定（21店舗 vs 16店舗）

## #46 よく使う仮説検定一覧（2標本）その2
URL: https://hsugaku.com/28-20

- **対応のある2標本の検定（対比較）**：
  - D = X_before - X_after として1標本の t 検定に帰着
  - H₀: μ_d = 0 vs H₁: μ_d > 0
  - 実例：ダイエットプログラムの効果検定（10名・t(9)）
- **2標本の比率差の検定**：
  - 正規近似：Z = (p̂ₐ - p̂ᵦ) / SE
  - 実例：動画サムネイルのクリック率（男性 vs 女性）

## #47 第1種の誤り・第2種の誤り・検出力
URL: https://hsugaku.com/29-10

| | H₀ が真 | H₁ が真 |
|--|--------|--------|
| H₀ 採択 | 正しい判断 | 第2種の誤り（β） |
| H₀ 棄却 | 第1種の誤り（α） | 正しい判断 |

- **第1種の誤り（α）**：H₀ が真なのに棄却する確率 = 有意水準
- **第2種の誤り（β）**：H₁ が真なのに棄却できない確率
- **検出力（Power）**= 1 - β：H₁ が真のとき正しく棄却できる確率
- α と β はトレードオフ（片方を下げると他方が上がる）

実例：第1種の誤り=0.2、第2種の誤り=0.4、検出力=0.6

## #48 p値
URL: https://hsugaku.com/29-20

- **p値の定義**：帰無仮説のもとで、検定統計量が実現値以上に極端な値をとる確率
- p値 < α → H₀ 棄却（有意）
- p値 ≥ α → H₀ 採択できない

片側・両側検定の違い：
- 片側検定のp値 = P(Z ≥ z₀)
- 両側検定のp値 = 2 × P(Z ≥ |z₀|)

実例：円盤直径の検定（n=10、μ₀=10、σ=0.5）
- Z ≈ 1.89、片側p値 = 0.0294、両側p値 ≈ 0.059

## #49 適合度検定
URL: https://hsugaku.com/30-10

- **目的**：観測度数が理論分布（仮定した確率分布）に適合するかを検定
- **統計量**：χ² = Σ(Fᵢ - npᵢ)²/(npᵢ) 〜 χ²(k-1)
  - Fᵢ：観測度数、npᵢ：期待度数、k：カテゴリ数
- 自由度 = (カテゴリ数 - 1)

実例：100個の1桁の数字が一様分布に従うか検定
- χ² = 19.4 > 16.92（棄却域）→ H₀ 棄却（偏りあり）

## #50 独立性の検定
URL: https://hsugaku.com/30-20

- **目的**：2つのカテゴリカル変数が独立（無関係）かを検定
- **クロス集計表（分割表）**を使用
- **統計量**：χ² = ΣΣ(Fᵢⱼ - fᵢ·f·ⱼ/n)²/(fᵢ·f·ⱼ/n) 〜 χ²((r-1)(s-1))
  - r：行カテゴリ数、s：列カテゴリ数
  - 期待度数 = 行周辺度数 × 列周辺度数 / 総計

実例：インフルエンザワクチン接種と発症の関係
- χ² = 32 > 6.63（1%有意水準）→ ワクチン効果あり
