## 二元配置分散分析の構造

一元配置分散分析が1つの因子（要因）の影響を検討するのに対し、**二元配置分散分析**は2つの因子を同時に扱う。例えば「薬の種類（A/B）」と「服用量（低/中/高）」が治療効果に与える影響を同時に評価する場合に使う。

モデルは：

$$Y_{ijk} = \mu + \alpha_i + \beta_j + (\alpha\beta)_{ij} + \varepsilon_{ijk}$$

ここで $\alpha_i$ は因子A（行因子）の主効果、$\beta_j$ は因子B（列因子）の主効果、$(\alpha\beta)_{ij}$ は**交互作用**、$\varepsilon_{ijk} \sim N(0,\sigma^2)$ は誤差項である。

## 変動の分解

全変動（SST）を以下の4成分に分解する：

$$SS_T = SS_A + SS_B + SS_{AB} + SS_E$$

| 変動源 | 自由度 | 平均平方 |
|--------|--------|----------|
| 因子A | $a-1$ | $MS_A = SS_A/(a-1)$ |
| 因子B | $b-1$ | $MS_B = SS_B/(b-1)$ |
| 交互作用 AB | $(a-1)(b-1)$ | $MS_{AB} = SS_{AB}/((a-1)(b-1))$ |
| 誤差 | $ab(n-1)$ | $MS_E = SS_E/(ab(n-1))$ |

各効果のF統計量は $F = MS_\text{effect} / MS_E$ であり、それぞれ対応するF分布で検定する。

## 交互作用の解釈

**交互作用**とは「因子Aの効果が因子Bの水準によって異なる」ことを意味する。交互作用がある場合、主効果のみを解釈することは誤りにつながる。

交互作用の有無は**交互作用プロット**（各因子Bの水準ごとに因子Aの平均値を折れ線グラフで描く）で視覚的に確認できる。折れ線が平行であれば交互作用なし、折れ線が交差または収束・発散すれば交互作用ありのサインである。

## 多重比較の問題（検定の多重性）

3水準以上の因子について「すべてのペア間で t 検定を繰り返す」と、第1種過誤（偽陽性）の累積確率が膨らむ（ファミリーワイズエラーレート, FWER の増大）：

$$1 - (1-\alpha)^m$$

$m$ 回の独立な検定で $\alpha = 0.05$ とすると、3ペアの比較（$m=3$）で FWER ≈ 14%、10ペアで FWER ≈ 40% にもなる。

## 多重比較の補正方法

**ボンフェローニ補正**：各検定の有意水準を $\alpha' = \alpha / m$ に設定する。単純で保守的（第1種過誤を厳しく制御する）が、検定数が多いと検出力が大幅に下がる。

**テューキー法（Tukey's HSD）**：すべての対比較に対して FWER を $\alpha$ に制御しつつ、ボンフェローニより検出力が高い。等サンプルサイズのすべてのペア比較に最適。

**シェッフェ法（Scheffé's method）**：事前に指定していない任意の対比（線形結合）の比較にも適用できる最も保守的な方法。事後的な探索的比較に向く。

使い分けの目安：
- 対比較のみ → テューキー法
- 任意の対比 → シェッフェ法
- 少数の特定比較 → ボンフェローニ補正

**落とし穴**：交互作用が有意な場合に主効果の解釈を優先する誤りが多い。交互作用が存在するときは因子を個別に解釈せず、水準の組み合わせ（セル平均）を直接比較すること（単純主効果の分析）が正しい手順である。

## Pythonでの活用例

```python
import pingouin as pg
aov = pg.anova(data=df, dv='score', between=['drug', 'dose'], ss_type=2)
posthoc = pg.pairwise_tukey(data=df, dv='score', between='drug')
```
