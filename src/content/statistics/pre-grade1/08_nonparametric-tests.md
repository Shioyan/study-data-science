## パラメトリック検定との違い

t検定やF検定などのパラメトリック検定は、データが特定の分布（正規分布など）に従うことを前提とする。これに対し**ノンパラメトリック検定**は分布の仮定を設けず、データの順位（rank）に基づいて検定を行う手法である。

ノンパラメトリック検定が適切な場面：
- サンプルサイズが小さく正規性の確認が困難
- データに外れ値が多い
- 順序尺度データ（「非常に満足・満足・普通・不満」など）
- 正規性の仮定が明らかに成立しない（強い歪み・外れ値）

## ウィルコクソンの符号順位検定（対応ある2標本）

対応ある2標本（同一対象への介入前後など）の中央値の差を検定する。t検定の代替として使われる。

手順：
1. 各ペアの差 $d_i = X_i - Y_i$ を計算
2. $|d_i|$ の大きさに順位をつける（$d_i = 0$ は除外）
3. 符号ごとに順位の和 $T^+, T^-$ を計算
4. 検定統計量 $W = \min(T^+, T^-)$ を用いて検定

中央値に差がなければ $T^+$ と $T^-$ は概ね等しいはずという直感に基づく。

## マン-ホイットニーの U 検定（対応なし2標本）

対応のない2グループの分布（または中央値）の差を検定する。t検定（2標本）のノンパラメトリック代替である。

統計量 $U$ は「グループ1の観測値がグループ2の観測値より大きい回数」を数えたものである：

$$U_1 = n_1 n_2 + \frac{n_1(n_1+1)}{2} - R_1$$

ここで $R_1$ はグループ1の順位の和（2グループ合算での順位）。帰無仮説は「2グループの分布が等しい」である。

## クラスカル-ウォリス検定（3群以上）

3群以上の独立標本を比較する一元配置分散分析のノンパラメトリック代替が**クラスカル-ウォリス検定**である。

全観測値を合算して順位をつけ、各群の平均順位が等しいかを検定する。帰無仮説を棄却した場合は多重比較（Mann-Whitney U 検定 + ボンフェローニ補正など）で事後検定を行う。

検定統計量：

$$H = \frac{12}{N(N+1)} \sum_{j=1}^k \frac{R_j^2}{n_j} - 3(N+1)$$

は近似的に自由度 $k-1$ のカイ二乗分布に従う。

## スピアマンの順位相関係数

2変数の単調関係の強さを測る指標が**スピアマン順位相関係数** $r_s$ である。各変数を順位に変換してピアソンの積率相関係数を計算したものと等しい：

$$r_s = 1 - \frac{6\sum d_i^2}{n(n^2-1)}$$

ここで $d_i$ は2変数の順位の差である。$|r_s|$ が 1 に近いほど強い単調関係がある。ピアソン相関が線形関係を捉えるのに対し、スピアマン相関は外れ値に頑健で非線形の単調関係も捉えられる。

## 検出力の低さとトレードオフ

ノンパラメトリック検定は分布の仮定を必要としない利点がある一方、パラメトリック検定が妥当な場合（正規分布データ）と比べると**検出力（統計的検出力）が低い**。これは順位変換により情報量が失われるためである。

正規分布データに対するノンパラメトリック検定の漸近相対効率（ARE）は t 検定との比較で約 0.955 であり、情報の損失は比較的小さい。

**落とし穴**：「外れ値があるからノンパラメトリック検定を使えばよい」という誤解がある。外れ値を除去すべきか、そもそもデータの性質が正規分布から外れているのかを先に検討すること。外れ値が測定誤差なら除去・修正、真の観測値なら外れ値に頑健な検定（あるいはノンパラメトリック検定）を選ぶ、という手順が正しい。

## Pythonでの活用例

```python
from scipy import stats
# Mann-Whitney U 検定
stat, p = stats.mannwhitneyu(group1, group2, alternative='two-sided')
# Spearman 相関
rho, p = stats.spearmanr(x, y)
```
