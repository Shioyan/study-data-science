## 主成分分析との根本的な違い

因子分析（Factor Analysis）は、観測変数の背後に潜む**潜在因子**（共通因子）を仮定し、観測変数をその因子で説明しようとする手法です。主成分分析（PCA）との本質的な違いを理解することが重要です。

| 観点 | 主成分分析（PCA） | 因子分析（FA） |
|------|-----------------|--------------|
| 方向性 | 観測変数 → 主成分（記述的） | 潜在因子 → 観測変数（説明的・因果的） |
| 目的 | 分散の最大化・次元削減 | 共分散構造の説明・潜在変数の発見 |
| 誤差 | 全分散を説明しようとする | 共通性（因子で説明できる部分）と独自性（誤差）に分ける |
| 解の一意性 | 固有値分解で一意に決まる | 回転の不定性がある（無限の解が存在する） |

「学力テストの各科目スコアの背後に"一般知能因子"が存在する」といった仮説を検証するのが因子分析の典型的な用途です。PCAはそのような「背後にある原因」の存在を仮定しません。

## 因子モデルの定義

$p$ 個の観測変数 $\mathbf{x}$ を $m$ 個の共通因子 $\mathbf{f}$ と固有因子 $\boldsymbol{\varepsilon}$ で表します：

$$\mathbf{x} = \boldsymbol{\mu} + \boldsymbol{\Lambda}\mathbf{f} + \boldsymbol{\varepsilon}$$

ここで $\boldsymbol{\Lambda}$（$p \times m$）は**因子負荷量行列**、$\mathbf{f} \sim N(\mathbf{0}, \mathbf{I})$、$\boldsymbol{\varepsilon} \sim N(\mathbf{0}, \boldsymbol{\Psi})$（$\boldsymbol{\Psi}$ は対角行列）と仮定します。

この仮定のもとで観測変数の共分散行列は：

$$\boldsymbol{\Sigma} = \boldsymbol{\Lambda}\boldsymbol{\Lambda}^\top + \boldsymbol{\Psi}$$

と分解されます。$\boldsymbol{\Lambda}\boldsymbol{\Lambda}^\top$ の対角成分を**共通性**（communality）、$\boldsymbol{\Psi}$ の対角成分を**独自性**（uniqueness）といいます。

- **共通性**：観測変数の分散のうち共通因子で説明できる割合（0〜1）
- **独自性**：その変数独自のばらつき（測定誤差＋固有変動）

## 因子負荷量の意味と解釈

因子負荷量 $\lambda_{jk}$ は、第 $k$ 因子と第 $j$ 観測変数の相関係数（直交回転後）に対応します。$|\lambda_{jk}|$ が大きいほどその変数がその因子と強く関連していることを意味します。

### 解釈の目安

| 因子負荷量の絶対値 | 解釈 |
|------------------|------|
| 0.7以上 | 非常に強い関連（実質的な指標変数） |
| 0.5〜0.7 | 強い関連（有意な指標変数） |
| 0.4〜0.5 | 中程度の関連（解釈に含める） |
| 0.3〜0.4 | 弱い関連（解釈は任意） |
| 0.3未満 | 無視してよい（その因子への負荷はほぼない） |

実務的には**0.4以上を有意な負荷の目安**とすることが多いです。

### 解釈の例

「語彙力」「読解力」「文法」という3変数の第1因子負荷量がすべて高ければ（例：0.8、0.75、0.7）、第1因子は「言語能力」と解釈できます。対して「計算」「図形」「論理」の第2因子負荷量が高ければ「数理能力」と解釈できます。

## 因子の回転

因子負荷量の解は一意でなく、直交変換の自由度があります。この自由度を利用して負荷量を「単純構造（各変数が一部の因子にのみ高い負荷量を持つ）」に近づける操作が**因子の回転**です。

### バリマックス回転（直交回転）で「単純構造」を目指す

**バリマックス回転（Varimax rotation）**は、各因子の負荷量の2乗の分散を最大化します。直感的には「各変数が特定の因子にのみ強く負荷し、他の因子にはほとんど負荷しない」状態（単純構造）を目指す操作です。

**単純構造のメリット**：
- 各因子が何を表しているかを解釈しやすい
- 「変数Aは因子1の指標、変数Bは因子2の指標」という明確な対応が得られる

バリマックス回転は因子間の無相関を保ちながら単純構造に近づけます。

### プロマックス回転（斜交回転）

因子間に相関を許す回転です。現実の潜在変数は多くの場合相関しているため（例：「言語能力」と「数理能力」は相関する）、より現実的なモデルとなります。ただし解釈は直交回転より複雑です。

## 因子数の決定方法

因子数 $m$ の選択には複数の基準があります：

1. **固有値 1 以上規準（カイザー規準）**：相関行列の固有値が 1 以上の因子を採用
2. **スクリープロット**：固有値の減少が緩やかになる点（肘）の前まで採用
3. **適合度検定**：モデルの適合度をカイ二乗検定で評価（最尤法での因子分析）
4. **解釈可能性**：因子に意味のある解釈が与えられる数を選ぶ（実践上最重要）

複数の基準を組み合わせて判断し、最終的には「各因子が実質的に解釈できるか」を優先させます。

## 確認的因子分析（CFA）と探索的因子分析（EFA）の使い分け

| 観点 | 探索的因子分析（EFA） | 確認的因子分析（CFA） |
|------|---------------------|---------------------|
| 目的 | データから因子構造を発見する | 事前に仮定した因子構造を検証する |
| 使うタイミング | 理論がなく、どんな因子が存在するか不明なとき | 既存理論に基づく仮説を新データで検証するとき |
| 変数と因子の対応 | 全変数がすべての因子に負荷（制約なし） | どの変数がどの因子に負荷するかを事前指定 |
| 適合度評価 | 主観的（解釈可能性） | CFI・RMSEA などの適合度指標で客観評価 |
| ソフトウェア | factor_analyzer (Python), factanal (R) | semopy (Python), lavaan (R) |

**正しい使い方**：EFAでデータを探索し、因子構造の仮説を立てたら、**別の独立したデータ**でCFAを実施して仮説を検証します。同じデータでEFAとCFAをどちらも実施することは過学習に相当し、避けるべきです。

## 因子スコアの解釈と利用

各観測値の因子スコアは以下で推定されます（回帰法）：

$$\hat{\mathbf{f}} = \boldsymbol{\Lambda}^\top \hat{\boldsymbol{\Sigma}}^{-1}(\mathbf{x} - \hat{\boldsymbol{\mu}})$$

因子スコアは後続の回帰分析やクラスター分析の入力として使われることが多いです。複数の指標変数をまとめて1つの潜在変数の得点として扱うことができます。

## Pythonによる実装例

```python
from factor_analyzer import FactorAnalyzer
import matplotlib.pyplot as plt
import numpy as np

# 1. 因子分析の実施（バリマックス回転）
fa = FactorAnalyzer(n_factors=2, rotation='varimax', method='ml')
fa.fit(df_scaled)

# 2. 因子負荷量の確認
loadings = fa.loadings_
print("因子負荷量行列:")
print(loadings)  # |0.4|以上の値に注目

# 3. 共通性の確認
communalities = fa.get_communalities()
print("\n共通性（各変数の説明率）:", communalities)

# 4. 固有値からカイザー規準で因子数を確認
ev, v = fa.get_eigenvalues()
print("\n固有値:", ev)

# 5. スクリープロット
plt.plot(range(1, len(ev) + 1), ev, 'ro-')
plt.axhline(y=1, color='blue', linestyle='--', label='カイザー規準')
plt.xlabel('因子番号')
plt.ylabel('固有値')
plt.title('スクリープロット（因子数の決定）')
plt.legend()
plt.show()
```

## 出力結果の読み方

- **因子負荷量行列**：各行が観測変数、各列が因子。0.4以上の値に注目して「この変数はこの因子に強く関連している」と解釈。
- **共通性**：0に近い変数はどの因子にも属さない「孤立した変数」であり、除外や再検討が必要。
- **独自性**：1 - 共通性。独自性が高い変数は共通因子で説明されにくく、独立した情報を持つ。

## 適用の前提条件

1. **十分な標本サイズ**：一般に変数数の10倍以上（最低でも100件以上）
2. **変数間に相関がある**：相関がなければ因子は存在しないため、まず相関行列を確認する
3. **正規性**（最尤法を使う場合）：多変量正規性を仮定
4. **線形性**：因子と観測変数の関係が線形

## 実務での落とし穴

- **EFAで見つけた因子を同じデータでCFAで検証する**：循環論法です。別サンプルで検証してください。
- **因子数を多く設定しすぎる**：因子が多すぎると解釈困難になります。解釈可能な因子数を優先してください。
- **負荷量が低い変数を無理に解釈する**：0.3未満の負荷量は「この変数はこの因子とは無関係」と判断してよいです。
