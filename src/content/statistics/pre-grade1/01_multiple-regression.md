## 重回帰分析とは

重回帰分析は、複数の説明変数（独立変数）$x_1, x_2, \ldots, x_k$ を用いて、1つの目的変数（従属変数）$y$ を予測・説明する手法です。単回帰が「1対1」の関係を捉えるのに対し、重回帰は現実のデータに潜む複雑な多変量関係を扱います。

モデルの基本形は以下のように表されます：

$$y = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \cdots + \beta_k x_k + \varepsilon$$

ここで $\beta_0$ は切片、$\beta_j$（$j=1,\ldots,k$）は各説明変数の偏回帰係数、$\varepsilon$ は誤差項（$\varepsilon \sim N(0, \sigma^2)$ を仮定）です。

## 最小二乗推定量（行列表現）

$n$ 個の観測値を行列でまとめると、モデルは $\mathbf{y} = \mathbf{X}\boldsymbol{\beta} + \boldsymbol{\varepsilon}$ と書けます。ここで $\mathbf{X}$ は $n \times (k+1)$ のデザイン行列（第1列がすべて1の切片列）です。

残差平方和 $S(\boldsymbol{\beta}) = (\mathbf{y} - \mathbf{X}\boldsymbol{\beta})^\top(\mathbf{y} - \mathbf{X}\boldsymbol{\beta})$ を最小化すると、最小二乗推定量が得られます：

$$\hat{\boldsymbol{\beta}} = (\mathbf{X}^\top \mathbf{X})^{-1} \mathbf{X}^\top \mathbf{y}$$

この推定量はガウス-マルコフの定理により、線形不偏推定量の中で分散が最小（BLUE: Best Linear Unbiased Estimator）となります。

### なぜ逆行列が出てくるのか

$(\mathbf{X}^\top\mathbf{X})$ はデータの散らばりと変数間の相関を要約した正方行列です。これを逆行列にすることで「データの相関を考慮しながら各変数の独立な効果を取り出す」操作をしています。直感的には「他の変数の影響を差し引いた純粋な効果」を数学的に実現するための手続きです。

## 偏回帰係数の解釈

$\beta_j$ は「他のすべての変数を一定に保ったまま $x_j$ が1単位増加したときの $y$ の変化量」を意味します。これが単回帰係数との決定的な違いであり、偏回帰係数は「他変数の影響を取り除いた純粋な効果」を表します。

### 数値例で理解する

年収（万円）を目的変数、年齢と経験年数を説明変数とした場合を考えます。

- 単回帰で「経験年数 → 年収」の係数を求めると、係数が大きく出ることがあります。
- しかし年齢と経験年数は強く相関しているため、「経験年数の効果」には「年齢の効果」が混入しています。
- 重回帰で両方を同時に投入すると、経験年数の偏回帰係数は「同じ年齢の人同士で比較したときの経験1年あたりの年収増加額」を意味します。

これが「偏（partial）」という言葉の意味です。他変数を偏らせずに固定した上での効果を見ているわけです。

## 標準化偏回帰係数（β係数）

非標準化偏回帰係数は変数の単位に依存します。「年齢（歳）」と「年収（万円）」では係数の大小を直接比較できません。そこで使われるのが**標準化偏回帰係数**（β係数）です。

$$\beta_j^* = \hat{\beta}_j \cdot \frac{s_{x_j}}{s_y}$$

ここで $s_{x_j}$ は $x_j$ の標準偏差、$s_y$ は $y$ の標準偏差です。β係数はすべての変数を標準化（平均0・分散1）した上で回帰したときの係数に等しく、単位によらず変数の「相対的な重要度」を比較できます。

| 係数 | 意味 | 使いどころ |
|------|------|----------|
| 非標準化偏回帰係数 | 元の単位での変化量 | 「1歳年齢が上がると年収が〇万円増える」など実務的解釈 |
| 標準化偏回帰係数（β係数） | 標準偏差単位での変化量 | どの変数が目的変数に最も影響力があるかの比較 |

## 決定係数と自由度調整済み $R^2$

モデルの当てはまりを評価する指標が決定係数 $R^2$ です：

$$R^2 = 1 - \frac{SS_\text{res}}{SS_\text{tot}} = 1 - \frac{\sum(y_i - \hat{y}_i)^2}{\sum(y_i - \bar{y})^2}$$

$R^2$ は説明変数を増やすだけで必ず上昇する欠点があるため、変数数を考慮した自由度調整済み $R^2$ が用いられます：

$$\bar{R}^2 = 1 - \frac{SS_\text{res}/(n-k-1)}{SS_\text{tot}/(n-1)}$$

モデル比較では $\bar{R}^2$ を使うのが原則です。$\bar{R}^2$ は変数の追加によってむしろ低下することがあるため、本当に有益な変数かを判断する指標になります。

## 多重共線性（VIF）

複数の説明変数が互いに強く相関している状態を**多重共線性（multicollinearity）**といいます。これが生じると $(\mathbf{X}^\top\mathbf{X})^{-1}$ の推定が不安定になり、偏回帰係数の分散が大きくなります。

### VIFの計算と意味

診断指標として**分散拡大係数（VIF: Variance Inflation Factor）**が使われます：

$$\text{VIF}_j = \frac{1}{1 - R_j^2}$$

ここで $R_j^2$ は $x_j$ を他の説明変数で回帰したときの $R^2$ です。

$R_j^2$ が高いほど $x_j$ は他の変数で説明できる＝多重共線性が強い、ということです。VIF はその「推定精度の悪化倍率」を示します。VIF = 5 なら「多重共線性がない場合と比べて係数の分散が5倍になっている」ことを意味します。

**VIF > 10 が問題とされる理由**：VIF > 10 は $R_j^2 > 0.9$ に相当し、変数の90%以上が他の変数で説明できる状態です。この場合、係数推定値が非常に不安定となり、符号が期待と逆転したり、標準誤差が巨大になったりします。

対処法：
- 高相関変数の一方を削除する
- 主成分回帰（PCA で多重共線性を解消してから回帰）
- リッジ回帰（正則化で係数の分散を抑制）

## 変数選択

最適なモデルを選ぶための情報量規準として AIC・BIC が広く使われます：

$$\text{AIC} = -2\ln L + 2k, \quad \text{BIC} = -2\ln L + k\ln n$$

BICはサンプルサイズ $n$ が大きいほど変数追加のペナルティが大きく、より簡潔なモデルを選びやすいです。

### ステップワイズ法の種類と注意

- **前進選択（forward selection）**：変数なしから始めてAICを改善する変数を順に追加
- **後退除去（backward elimination）**：全変数から始めてAICが悪化する変数を順に除去
- **変数増減法（stepwise）**：追加と除去を繰り返す

ステップワイズ法は多重検定の問題があるため、AIC/BIC 規準と組み合わせて使うのが望ましいです。「AICを最小化する」方向でステップを進めるのが標準的な実装です。

## Pythonによる実装例

```python
import statsmodels.api as sm
import pandas as pd
from statsmodels.stats.outliers_influence import variance_inflation_factor

# モデルの推定
X = sm.add_constant(df[['x1', 'x2', 'x3']])
model = sm.OLS(df['y'], X).fit()
print(model.summary())  # 偏回帰係数・t値・p値・R²を一覧表示

# VIFの計算
vif_data = pd.DataFrame({
    'feature': X.columns,
    'VIF': [variance_inflation_factor(X.values, i) for i in range(X.shape[1])]
})
print(vif_data)  # VIF > 10 の変数は多重共線性に注意

# AICによるステップワイズ変数選択（forward）
from statsmodels.regression.linear_model import OLS
import itertools

def stepwise_aic(df, target, candidates):
    selected = []
    best_aic = float('inf')
    for feat in candidates:
        X_try = sm.add_constant(df[selected + [feat]])
        aic = OLS(df[target], X_try).fit().aic
        if aic < best_aic:
            best_aic = aic
            best_feat = feat
    selected.append(best_feat)
    return selected
```

## 出力結果の読み方

`model.summary()` の主要な数値の見方：

| 出力項目 | 意味 | 判断基準 |
|----------|------|--------|
| `coef` | 偏回帰係数 | 符号と大きさで効果の方向と強さを判断 |
| `t` | t統計量 | 絶対値が2以上でおおむね有意 |
| `P>|t|` | p値 | 0.05未満で有意（5%水準） |
| `R-squared` | 決定係数 | 高いほど当てはまりが良いが過信は禁物 |
| `Adj. R-squared` | 調整済みR² | モデル比較にはこちらを使う |
| `AIC` | 赤池情報量規準 | 小さいほど良いモデル |

## 適用の前提条件と診断

重回帰分析の推定値が信頼できるためには、以下の前提条件を確認する必要があります。

1. **線形性**：$y$ と各 $x_j$ の関係が線形である。残差 vs 予測値プロットで確認。
2. **独立性**：観測値が互いに独立である。時系列データでは自己相関に注意。
3. **等分散性（ホモセダスティシティ）**：誤差の分散が一定である。残差プロットが水平なバンド状であることを確認。
4. **正規性**：誤差項が正規分布に従う。QQプロットで確認。
5. **多重共線性なし**：VIF で確認。

## 実務での落とし穴

- **$R^2$ の高さだけでモデルを評価してはいけません**。説明変数をやみくもに増やすと訓練データへの過学習が起きます。必ず $\bar{R}^2$ や AIC で評価し、残差プロットで仮定を確認してください。
- **交絡変数を見逃さない**：重要な説明変数を省略すると「欠落変数バイアス（omitted variable bias）」が生じます。ドメイン知識をもとに変数を選ぶことが重要です。
- **外挿の危険性**：学習データの範囲外への予測は信頼できません。
